Jede Aufgabe hat ihren eigenen Ordner, dieser ist folgendermaßen strucktuiert:
* Augabe

    * Ausführbare Datei

    * Dokumentation (pdf - Datei)
    * optional Beispielbilder

    * Quelldateien
