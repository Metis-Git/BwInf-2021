## Struktur

Jede Aufgabe hat ihren eigenen Ordner, dieser ist folgendermaßen strukturiert:

* Aufgabe
  * bin
   * ausführbare Datei
  * resources (optional)
   * Bilder etc.
  * src
   * Quelldateien

Die Dokumentation der Aufgaben sind wie die Aufgaben benannt und befinden sich im Hauptordner.
